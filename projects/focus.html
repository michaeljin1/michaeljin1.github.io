세션 평가 흐름

1
의도 선언 + 스톱워치 시작 ~0s
Today 패널에서 Todo의 ▶ 클릭 → 카테고리 자동 매핑 (예: "Coding") + Today 윈도우 자동 접힘.
2
행동 신호 측정 (지속) 실시간
ActivityWatch가 활성 윈도우·앱·AFK를 1초 단위 폴링. 화면 픽셀·키스트로크·URL 일절 수집 X.
3
세션 종료 + 데이터 청크 ~0s
사용자가 [완료] / [종료] 클릭 → 스톱워치 정지 → 세션 시간 범위로 AW raw event 청크.
4
카테고리 집계 ~30ms
17 카테고리 × 6 그룹 룰로 raw event → episode 변환. 의도 vs 비의도 카테고리 시간 분리.
5
4차원 품질 점수 (Rule) ~5ms
Alignment · Depth · Resistance · Stability를 가중 합 (0.40·0.25·0.20·0.15)으로 0~100 즉시 환산.
6
패턴 인사이트 (LLM, 비동기) ~2s · 일 1회
하루 마감 시 Gemini Flash가 카테고리·앱·점수만(=익명화) 받아 자연어 인사이트 생성. 점수는 Rule, 이야기는 LLM.
