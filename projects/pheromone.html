Pheromone Lab

Claude Code sub-agent 통신 모델 3-way 비교 실험.

Enhans의 Pheromone Protocol(개미 군집 지능 모델)을 Claude Code sub-agent 시스템에 직접 구현. sub-agent끼리 직접 호출하지 않고 공유 마크다운 파일에 신호를 남기면 다음 단계가 반응합니다. 동일 환경에서 기존 방식(Ad-hoc 단발 호출, Relay 응답 릴레이)과 정량 비교 실험을 진행해 메인 컨텍스트 5.6배 경량화 / 영속 추적 / 자가 회복 사이클을 데이터로 입증했습니다.

이점 · 비용 · 성과

이점
  • 1회 명령만으로 자율 실행
  • 각 단계에서 높은 품질의 산출물 생성
  • 로그 누적 및 재시작 가능
비용
  • 느린 작업 시간
  • 횡단 검증 불가
  • 작업 과잉
성과
  • 긴 호흡의 작업을 지시할 때 사용
  • 내용에 따라 필수 agent만 채택하도록 최적화
