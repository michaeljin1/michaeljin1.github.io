영상 처리 흐름

1
자막 추출 ~6s
youtube-transcript-api + Webshare proxy. 실패 시 재시도 3회 → yt-dlp 폴백 → api.list() 자동 선택.
2
청크 분할 <100ms
최대 3,000자 + 15% overlap. 6,000자 영상 → 약 2~3개 청크.
3
청크별 LLM 분석 (병렬) ~10~15s
Gemini Flash Lite × 8 동시 (Semaphore). section_title · summary · concepts · best_quote · density 추출.
4
brief 임베딩 ~150ms
title + tags + concepts + brief → Gemini embedding-001 → 768d 벡터.
5
분리수거 판정 ~50ms
pgvector RPC로 라이브러리 top-1 cosine 비교. raw ≥ 0.85 → skip / 0.72~0.85 → partial / < 0.72 → must-watch.
6
캐시 + 저장 ~200ms
video_id 기준 글로벌 캐시. 같은 영상 재분석 시 0 API 호출 (25s → 0.5s). concept 자동 dedup (cosine ≥ 0.85).
